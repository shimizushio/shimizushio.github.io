一吐为快：离婚冷静期

发表于 2020-05-29 更新于 2020-05-30 分类于 牢骚 Waline：

“离婚冷静期”，它来了。2021年1月1日开始实行。

先说我个人的想法，那就是在这玩意被废除之前，能让我考虑结婚的最低条件就是“不是中国国籍”。嗯没错最低条件，别的条件合不合另外再说，但不能满足这条就绝不考虑。理由也很简单，一看就明白了吧，我不希望如果哪天我真结婚了然后发现对方是个渣男，还要受“离婚冷静期”的制约。被法律专业的亲友指正了，不受制约的条件只要登记地不受中国婚姻法管辖就行……

两个人在一起本来就是抽卡，而当今世下能抽到普通卡的几率都已经太低，放眼望去所见的男人大多都是那个鬼样子，家暴的家暴，杀人的杀人，甚至于只要不动女方一下都能俨然得到“好男人”的头衔。我自认我足够了解的几位男性朋友，也是因为长年的相处，我知道了对方的为人——但在这里冒犯他们一下，我得说句难听的，我也不能保证我经过长年的时间所了解到的就是对方的真面目。

连长年相处的亲友都不过如此，随便找个不认识的男人相亲结婚不就是更可怕的事情了吗。然后连撕卡都不让撕，还要加个离婚冷静期，那于我而言最好的选择便是直接不抽这个卡池。

如何看待离婚冷静期拟写入民法典分编草案？ - 知乎

虽然高中时期我曾一度坚信自己就是同而否决掉其它所有可能性的思考回路，现在回过头去看十分可笑——但随着这玩意的一锤定音，我不禁回头看了看高中时的自己，或许……

我一直还在潜意识中觉得自己的性取向不是能拿出来大庭广众地说的东西，但既然是个人博客，说一说也没什么问题吧，反正不会有人来看：也许…和一个女孩子在一起可能真的比抽卡抽个男人结婚要好得多、简单得多。

发表于 2020-05-28 更新于 2020-05-29 分类于 杂谈 Waline：

心血来潮，到神奇宝贝百科的当年自己用户的讨论页看了一下，居然到最近还有人问我在不在噗哧……

还有人说我是因为高考的原因不上线…嘛，你们怎么想都好了

对我来说只是兴趣转移了而已…我想想2015年的时候我兴趣在哪来着…噢好像在DNF（？）

实话说当时好像还有什么更深刻的理由不去神奇宝贝百科，但我完全没有记忆了。既然想不起来，就当它是已经被遗忘的过去好了。

也不是没有想过回去写点什么，但现在真的既没有兴趣也没有时间了，FF14和明日方舟真的能吃掉我所有的时间，现在要去百科写点什么的话可能也更愿意写FF14相关的内容不了不了，比起写百科我还是更愿意写同人文。还有就是和一些没长大的小孩子交流也很费劲……或者不如说当年的自己也是因为没长大，太过幼稚，所以才会在神奇宝贝百科编辑……？

怎么说呢这个话题。

我当时还真的天真地以为自己会一直有热情，能一直把编辑的工作做下去……不过17的事情也可以看出我其实一点都没有热情，只知道开坑不知道填坑。也算是幼稚的一种吧，如果能更有规划一点，或许就不会这样了。

以及，我真的很厌烦宝可梦 剑盾。按我的观点来看，Game Freak要么是掉钱眼里了，要么就是从一开始就在钱眼里没爬出来过。作为一个商业公司，其实无可厚非，我无非只是感叹一下当初的自己还是太过幼稚，错爱了这个以为能陪伴我走下去的作品。不过我想说的其实是，我既然对宝可梦的最新作没有任何兴趣，那说明我就算真的回去了神奇宝贝百科，也没什么好参与编辑的就是了。

不如就像现在这样默默守望就好了。对于我拉入坑的新宝可梦玩家们，我还是会推荐神奇宝贝百科的，作为一个Wiki站，神奇宝贝百科做得真的很不错，只不过那和我毫无关系罢了。

万一，万一真的有神奇宝贝百科的用户们能找到这里的话……算了啦，没有那种万一。再说了，除了水跃阿布超龙格莱Leo之外谁认识你啊。

……要是真的你们能找过来的话，可以在这里的评论里留个脚印？当然不愿意的话倒也无所谓。

不过我还是会当你们找不到这里然后毫无顾忌地在这里诉说过往的记忆的——指我还能记得的部分。

发表于 2020-05-22 更新于 2020-05-25 分类于 追忆 Waline：

高中时期的我倒是十分确信自己是个同（…

也不知道那个时候哪来的自信就给自己下的判断书。现在想来当初可能只是因为我喜欢的二次元角色大多都是女孩子所以让我真的产生了那样的错觉……或者我其实是清楚究竟是怎么一回事的，只是在欺骗自己，对自己装傻。

不管怎么说，现在我早已经不是当时那个懵懂的高中生而成为了一个没人要的老阿姨。

二次元这个词可能在现如今的中文语义里衍生出了不太妙的更多含义，但…你们能懂我想表达的意思就行。

那时候还有个叫做夜莺的孩子陪我玩过家家游戏，现在回想起来那可真是羞耻。聊天记录我甚至都有存，但那可不是能放出来的级别——我自己都无法直视那个打开的txt文件，所以除非夜莺小姐本人跑出来放记录，否则不会有任何人能再看到那时候的黑历史的具体内容。

但我与夜莺小姐也失联快要数十上百年了罢，我更不相信她可能会回来。对我是强烈的黑历史的同时，想必对她来说也是。

不如说，正因为有她不可能看到这里的自信，我才敢在这里偷偷提起她。

我或许没有说过我开始写同人的契机。现在回想起来还是难以理解，我究竟是为何对《地下城与勇士》这款游戏产生写同人的想法，而且还是写魔法师贴贴的百合小故事的想法。一口气打开了一大堆不该打开的大门呢。

尽管数年过去，别的不说，笔力完全没有进步就是了。

夜莺便是我在那时候，在DNF同人那个圈子里遇到的孩子。尽管我说我与她失联甚久，但她应该是在那个圈子里与我保持联系最久的孩子了。其他的人们我现在也记得不少，但恕我不能在这里报上她们的名字与ID。我偷偷去看过她们，她们各自都在自己的新天地里活得很开心，也许早已忘了DNF是什么。我和她们虽说认识，但也并未有多熟络，恐怕我现在向她们报上那时候的名字，也只能得到“那是谁？”的回答了罢。我不该去打扰她们。

本来还想多说些和夜莺的小故事，但那过家家的内容实在是太羞耻了，不想再提。不过我后来又打开语C的大门，我想也要归功于和她的过家家吧。

虽然现在很羞耻，但当时的我玩得可开心了，这个过家家持续的时长可是以年来计的。

然后，她就突然消失在了我的世界里。或许是兴趣转移了，或许是有什么别的原因。对于她不是渐渐少话而是突兀地消失，我也有担心过是不是因为我的缘故，只是一切的一切都随着她突兀的消失戛然而止，我无论是再抱有什么样的疑问埋着什么样的想法，都不再能和她诉说了。

那，就这样把我心中的念想也强硬斩断，也没什么不好。

只是在《明日方舟》里看到这个叫做Nightingale的角色的时候，我有些惊讶就是了。恰巧的同名，而且Nightingale的角色立绘和人设实在和她有太多的共同点，就连那温柔的语句，安在当年的那只夜莺头上也毫无违和。她不禁唤醒了我尘封已久的记忆。

但我清楚的，她终究不是她。

我不知道她是否仍安好，也无处向她确认。我只能但愿我希望她安好的这个愿望，能穿越可能需要穿越的时空、跨过可能需要跨过的世界的边境，到达她的身旁。

真是看清了人性呢。

日本给每个人发的10万日元补助金，因为是家庭代表去申请和领取的，所以就有丈夫领了全家的50万日元然后宣称全部自己用这样的恶劣行径。

至于家庭代表能不能是妻子呢，这个我不知道，但应该一开始在市政厅登记的是谁就是谁，并不是随便一个人都可以的，而登记丈夫的人多还是登记妻子的人多，我想这是一件不言自明的事情。

简直活生生的反婚广告，不是么？只要不结婚，就不会被丈夫夺走本该是自己的补助金。

还有家长直接拿走孩子的补助金的事情，简直是贻笑大方呢。

不过根据这位律师的见解来看，这份补助金的发放对象其实就是家庭代表，而其他人只不过是被当作发放补助金时要计算的人头数而已。家庭代表当然可以擅自使用所有的补助金而无需按人头分配。

这不是太棒了吗，简直是活生生的反婚广告啊。既然我不是家庭代表的时候我就只是个人头，那我干嘛还要结婚呢，只要我独身，家庭代表就会是我自己，因为家庭里就我一个人，自然就不会有问题。

……说来说去，说到最后不还是婚嫁之事……

悲哀的是，不管我的言辞再怎么激动，我看到莲大人和她的男友恩爱的时候还是会羡慕嫉妒…我不觉得我能找到合适的男友。或者其实可以把范围扩大一些，我不觉得我能找到合适的男友或女友。

……啊关于这个事我以后可能会另说的啦。其实我也不清楚现在的自己究竟是同是双还是异꒰ˊ• ·̭ •̥ ꒱不过那不是很重要。

只要知道找不到就对了。

可能世人会觉得那些渣男事例过去了就过去了没什么影响，但至少于我而言，就算我忘记了事例本身说的是谁，但一点点打起的退堂鼓却不会因为事情过去了就偃旗息鼓。我对于自己能找到合适的另一半的信心…一开始就没抱什么信心，不过确实每看到一个家暴案例，每看到一个令人痛心的案子，我就会退一步，最后到了现在已经完全没有去找的想法了——我不可能提前在熟起来之前就知道对方是不是渣，而这么多案例告诉我如果我真的在人堆里抽奖，我抽中渣的概率其实相当之高。

这太可怕了，还是一个人活下去吧。不抽奖也不会死嘛。

腆着脸被骂两句老大不小了怎么还不结婚，总比结了婚被家暴求助无门要好得多吧…
