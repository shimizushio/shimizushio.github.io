【FF14】鱼王分解笔记

2020-05-30
笔记

感觉最近饿猫总是上不去…自己做个笔记记录一下好了

记录的是需要分解鱼（基本上都是鱼王）才能产出的幻化

分解目标全是新生篇鱼王，至少90装等以上再来！当然最好是满级技能全了再来钓会更简单～

上衣

ギガシャーク

  • Lodestone（ウェーブサマーホルター）

  • 灰机Wiki（浪花女式夏衣）

  • Lodestone（サザンシー・タリスマン）

  • 灰机Wiki（南海护身符）

  • 分解目标：ギガシャーク（巨鲨）

  • 垂钓地点：中央ラノシア サマーフォード沿岸（断崖下的西侧海岸）

  • 鱼饵：フローティングミノー

  • 天气：快晴･晴れ

  • 垂钓途径：

    • フローティングミノー → メルトールゴビー（弱震） → ギガシャーク（激震）
    • フローティングミノー → ハーバーヘリング（強震） → ギガシャーク（激震）
    • フローティングミノー → ハーバーヘリング（強震） → オーガバラクーダ → ギガシャーク（激震）

ラブカ

  • Lodestone（コロナルサマーホルター）

  • 灰机Wiki（盛花女式夏衣）

  • Lodestone（ティ・リーフ・レイ）

  • 灰机Wiki（花草环）

  • 分解目标：ラブカ（皱鳃鲨）

  • 垂钓地点：西ラノシア 船の墓場（幻影群岛南东）

  • 鱼饵：ラグワーム

  • 天气：曇り･霧･風

  • 时间：ET 17：00～02：59

  • 垂钓途径：

    • ラグワーム → メルトールゴビー（弱震） → ワフー（強震） → ジャイアントスキッド（激震） → ギガシャーク（激震）

下装

メガオクトパス

  • Lodestone（ウェーブサマーパレオ）

  • 灰机Wiki（浪花女式海滩裙）

  • Lodestone（ブルーサマーマロ）

  • 灰机Wiki（海蓝男式海滩裙）

  • 分解目标：メガオクトパス（猛犸章鱼）

  • 垂钓地点：リムサ・ロミンサ：下甲板層

  • 鱼饵：ピルバグ

  • 时间：ET 09：00～16：59

  • 垂钓途径：

    • ピルバグ → メガオクトパス（激震）
    • ピルバグ → メルトールゴビー（弱震） → メガオクトパス（激震）
    • ピルバグ → ハーバーヘリング（強震） → メガオクトパス（激震）
